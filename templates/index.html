Project Description

This project showcases a complete end-to-end MLOps pipeline. It utilizes Cloudflare R2 storage for trained model persistence and MongoDB for real-time data retrieval, enabling continuous updates for training and testing. GitHub Actions are integrated for CI/CD deployment, and Render is used to deploy the functional application.

For a comprehensive understanding of the project's workflow and detailed documentation, please refer to the README file in the repository.
